# 介绍
这是深知的智能体部分。报告生成功能参考了WebThinker的"搜索-思考-写作"策略 (搜索还没做)；PDF总结功能可以总结多个PDF的大致内容
# Quick Start
### 报告生成
python test.py
### PDF总结
python test_PDF.py
# 详细说明
1. 报告生成、PDF总结功能运行后，都会在ouput目录下生成.md报告。(报告是流式生成的，模型输出会实时追加到报告中)
2. 配置请在config.py中修改，里面有详细的注释说明
3. 我的智能体暂时部署在线上，方便其它同学快速通过代码调用API来使用，会消耗我的token额度，所以请不要高强度地使用API。（智能体部署在Dify上，Dify部署在Docker上，如果另一位同学要本地部署智能体的话要配很多的环境，还要注册多种API，很不方便）
4. 智能体使用的模型是Qwen2.5-Instruct-32B
